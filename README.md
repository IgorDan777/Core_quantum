				### Краткое описание для файла `README.txt` (квантовый симулятор)


							**Название проекта**  
Quantum Simulator — простой симулятор квантовых вычислений.


								**Описание**  
Программа моделирует работу квантового компьютера с ограниченным числом кубитов. Поддерживает:
- базовые квантовые гейты (`X`, `H`);
- двухкубитный гейт `CNOT` (в базовом режиме);
- упрощённую версию алгоритма Шора (в режиме `shor`).


Предназначена для обучения и демонстрации принципов квантовых вычислений.


								**Режимы работы**
1. **Базовый** (`basic`, 6 кубитов):  
   - Гейты: `X`, `H`, `CNOT`.
   - Подходит для изучения суперпозиции и запутывания.
2. **Упрощённый Шора** (`shor`, 10 кубитов):  
   - Демонстрация начального этапа алгоритма Шора (применение `H` ко всем кубитам).


								**Основные возможности**
- Применение квантовых гейтов к отдельным кубитам.
- Визуализация вероятностей состояний.
- Переключение между режимами.
- Сброс состояния в `|00...0⟩`.


								**Как начать**
1. Запустите скрипт:  
   ```bash
   python quantum_simulator.py
   ```
2. Выберите режим (`1` — базовый, `2` — Шора).
3. Вводите команды (см. `help`).


								**Доступные команды**
- `h <qubit>` — гейт Хадамара на кубите `<qubit>`.
- `x <qubit>` — гейт Паули‑X на кубите `<qubit>`.
- `cnot <c> <t>` — CNOT: control `<c>`, target `<t>` (только в `basic`).
- `shor` — запустить упрощённый алгоритм Шора (только в `shor`).
- `reset` — сбросить состояние.
- `state` — показать полное состояние.
- `print` — вывести состояния с вероятностью > 1e‑6.
- `visualize` — графически показать вероятности (p > 0.001).
- `mode <basic|shor>` — сменить режим.
- `help` — справка.
- `exit` — выход.


								**Требования**
- Python ≥ 3.6;
- библиотеки: `numpy`, `matplotlib`.


								**Пример сессии**
```
> h 0        # Создаём суперпозицию на q₀
> cnot 0 3   # Применяем CNOT
> print      # Смотрим результаты
```
